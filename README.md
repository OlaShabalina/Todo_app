# Todo_app
Making a task manager for adding and filtering tasks for the day (todo app). 

### Languages used
HTML, JavaScript

### Styles 
Materialize front-end framework and css.

### Features

* Filter tasks
* Add a new task
* Complete task
* Delete task
* Hide completed tasks
